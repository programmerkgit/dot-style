# dot-style
Simple styling class helpers for adjusting styles.

## Quick start
Several start options

- Clone the repo: `git clone git@github.com/dot-style.git`
- Install with [npm](https://www.npmjs.com/): `npm install dot-style`

##Get Staretd
You add css file `dot-style/dist/dot-style.css` to you project.


##usage

### Dimension
You can style padding, margin with classes.

####Padding
```
<p class="pt-30px"></p>
<p class="pr-30px"></p>
<p class="pb-30px"></p>
<p class="pl-30px"></p>
<p class="px-30px"></p>
<p class="py-30px"></p>
<p class="p-30px"></p>
```

####Margin
```
<p class="pt-30px"></p>
<p class="pr-30px"></p>
<p class="pb-30px"></p>
<p class="pl-30px"></p>
<p class="px-30px"></p>
<p class="py-30px"></p>
<p class="p-30px"></p>
```
